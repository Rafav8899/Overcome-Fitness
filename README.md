# Overcome Fitness 💪

Landing page para un emprendimiento de Personal Trainers.

## ✨ Características
- Diseño mobile-first
- Totalmente responsive
- Header que se oculta al hacer scroll
- Estética moderna y elegante
- HTML, CSS y JavaScript puro

## 🛠 Tecnologías
- HTML5
- CSS3 (variables, flexbox, grid)
- JavaScript (vanilla)

## 🎯 Objetivo
Proyecto frontend enfocado en experiencia de usuario y presentación visual,
pensado como landing real para un emprendimiento.
